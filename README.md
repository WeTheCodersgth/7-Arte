# 7-Arte
Plataforma de streaming de filmes  (PWA) com funcionalidades de listagem, busca e filtragem. O sistema  incluirá uma área administrativa para gestão de conteúdo (filmes,  categorias, etc.), autenticação segura para administradores e usuários, e  um reprodutor de vídeo integrado. O objetivo é testar a performance,  sincronia e colaboração da equipa.
